# EAbotoy-plugins

# 配置

新建`EAbotoy.json`文件进行配置

```json
{
  "host": "http://127.0.0.1",
  "port": 8888,
  "wxid": "机器人wxid号",
  "master": "你的wxid号",
  "env": "dev",
  "group_blacklist": [],
  "friend_blacklist": [],
  "blocked_users": [],
}

```

# 运行

无特殊说明，说明插件可以直接运行

```shell
pip install -r requirements.txt
python boy.py
```
